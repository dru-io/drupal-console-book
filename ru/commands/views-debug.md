# views:debug
The **views:debug** command Display current views resources for the application

**Использование:**
```
$ drupal views:debug [arguments] [options] 
$ vde  
```

## Доступные опции
Опция | Описание
-------|-------------
--tag | View tag
--status | View status (Enabled|Disabled)

## Доступные аргументы
Аргумент | Описание
---------|-------------
view-id | View ID
