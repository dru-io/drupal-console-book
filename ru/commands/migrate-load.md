# migrate:load
The **migrate:load** command Generate a migration entity.

**Использование:**
```
$ drupal migrate:load [arguments] [options] 
```

## Доступные опции
Опция | Описание
-------|-------------
--override | The File is already installed, do you want override it [yes/no] ?

## Доступные аргументы
Аргумент | Описание
---------|-------------
file | Path to the yml file
