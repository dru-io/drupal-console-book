# cache:rebuild
**cache:rebuild** команда очищает и пересобирает все кэши сайта.

**Использование:**
```
$ drupal cache:rebuild [arguments]
$ cr  
```

## Доступные аргументы
Аргумент | Описание
---------|-------------
cache | Очистить только указанный кэш

## Примеры
* Очистить все кэши
```
$ drupal cr all
```
* Очистить кэш discovery
```
$ drupal cr discovery
```
